### How To Run

#### preparetion

* install the aws-cli

* run `aws-config`, then input the key id and key secret


#### linux / macOS

* create the stack
```
create.sh
```

* update the stack
```
update.sh
```

* delete the stack
```
delete.sh
```

#### Windows

* create the stack
```
create.bat
```

* update the stack
```
update.bat
```

* delete the stack
```
delete.bat
```
